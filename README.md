# Customer_behaviour
Data analytics project1 (customer behaviour analysis)
Overview

This project demonstrates an end-to-end data analytics workflow, starting from loading a dataset in Python to creating an interactive Power BI dashboard and generating analytical insights. The goal of the project is to explore the dataset, clean and prepare the data, perform exploratory analysis, and present key findings through visualizations and reports.

**Dataset**

The dataset used in this project contains structured data used for analysis and visualization. It includes multiple variables that allow exploration of trends, patterns, and relationships.

**Key characteristics:**
Structured tabular data
Multiple numerical and categorical features
Suitable for exploratory analysis and dashboard visualization


**Tools & Technologies**
The following tools were used in this project:
      Python – Data loading and analysis
      Pandas – Data cleaning and manipulation
      NumPy – Numerical operations
      Matplotlib / Seaborn – Data visualization for EDA
      Power BI – Interactive dashboard creation
      Jupyter Notebook – Analysis and documentation

**Project Workflow**
The project follows a standard data analytics pipeline:
Data Loading
    Imported dataset using Python
    Inspected structure, data types, and basic statistics

Exploratory Data Analysis (EDA)
    Examined distributions and relationships
    Identified trends, patterns, and anomalies
   Created visualizations to better understand the data

Data Cleaning
    Handled missing values
    Removed duplicates
    Corrected data types
    Prepared the dataset for analysis and visualization

Dashboard Development
    Imported cleaned dataset into Power BI
    Created interactive visualizations
    Designed a dashboard to highlight key insights

Reporting
    Summarized findings from the analysis
    Documented insights and observations


**Dashboard**
     The Power BI dashboard provides an interactive way to explore the data and insights.

Key features:
Interactive filters and slicers
Trend visualizations
Summary metrics and KPIs
Comparative charts for better decision making

**Results & Insights**
Through data analysis and visualization, the project identifies important patterns and insights from the dataset. The dashboard enables users to quickly understand trends and make data-driven decisions
